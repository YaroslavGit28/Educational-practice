# DEMO_GUIDE.md

## Как быстро проверить проект

1. Запустите `scripts\setup.bat` (первый раз) или `scripts\run.bat`
2. Откройте http://127.0.0.1:8000/
3. Выполните основной сценарий:

### Сценарий покупателя

1. Войти: `customer@shop.ru` / `demo1234`
2. Перейти в каталог → добавить товар в корзину
3. Оформить заказ
4. Проверить «Мои заказы»

### Сценарий менеджера

1. Войти: `manager@shop.ru` / `demo1234`
2. Панель менеджера → подтвердить заказ

### Сценарий кладовщика

1. Войти: `warehouse@shop.ru` / `demo1234`
2. Начать сборку → завершить отгрузку

## Тестовые данные

| Логин | Пароль | Роль |
|-------|--------|------|
| admin@shop.ru | demo1234 | Администратор |
| customer@shop.ru | demo1234 | Покупатель |

## Успешная проверка

- Приложение открылось
- Авторизация работает
- Заказ создаётся и меняет статус
- `/api/health/` возвращает `status: ok`
