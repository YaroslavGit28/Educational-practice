# RELEASE_NOTES.md

## Релиз 0.1.1

### Что изменилось

Проект подготовлен к сопровождению по УП.03: добавлены скрипты запуска, Docker, тесты, документация, CI/CD и исправления безопасности.

### Что исправлено

- Ошибка редиректа после авторизации с параметром `next`
- Проблемы запуска проекта (путь к apps, wsgi, миграции)
- Отсутствие шаблонов и пустые модули admin_panel/reports

### Как проверить

1. `scripts\setup.bat`
2. `scripts\run.bat`
3. Открыть http://127.0.0.1:8000/
4. Войти как `customer@shop.ru` / `demo1234`
5. Оформить тестовый заказ

### Ограничения

- SQLite (не для высоконагруженного production)
- HTTPS не настроен (учебный проект)
- PaaS-деплой не выполнялся (использован Docker demo-стенд)
