# Проблема: некорректный редирект после входа

## Описание

При авторизации с параметром `next` в URL происходит ошибка редиректа, если передано имя маршрута вместо пути.

## Шаги воспроизведения

1. Открыть `/users/login/?next=catalog:index`
2. Ввести email и пароль
3. Нажать «Войти»

## Ожидаемый результат

Перенаправление на главную страницу или указанный URL.

## Фактический результат

Ошибка NoReverseMatch / некорректный redirect.

## Окружение

- ОС: Windows 10
- Python: 3.12
- Django: 4.2.7
- Режим: local
