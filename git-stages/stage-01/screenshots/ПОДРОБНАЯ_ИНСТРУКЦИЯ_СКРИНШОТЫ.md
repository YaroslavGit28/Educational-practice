# Подробная инструкция по скриншотам для УП.03

Скриншоты — обязательная часть сдачи **каждого этапа**. На каждом снимке должно быть видно **конкретное действие и результат**, а не рабочий стол без контекста.

## Общие правила

1. **Формат:** PNG или JPG, читаемый текст.
2. **Имена файлов:** строго как в методичке (`01_repo_structure.png`, не `скрин1.png`).
3. **Куда класть:** `screenshots/stage-N/` (для этапа N) или папка сдачи по методичке колледжа
4. **Как снимать:**
   - Windows: `Win + Shift + S` → выделить область → сохранить в нужную папку
   - Или `PrtSc` / «Ножницы»
5. **На скрине должно быть видно:**
   - окно терминала с **командой** и **результатом**
   - или браузер с **адресной строкой** (URL)
   - или редактор с **путём к файлу** в заголовке
6. **Не размывайте** пароли — в `.env.example` их и так нет.

## Про ошибку pip при setup.bat

Если видите `ReadTimeoutError` к pypi.org:

1. Проект **уже может работать** — в `.venv` Django установлен.
2. Запустите напрямую:
   ```bat
   .venv\Scripts\python.exe manage.py runserver
   ```
3. Обновлённый `setup.bat` использует `.venv\Scripts\python.exe` и не лезет в системный Python.
4. Для скрина `02_install_dependencies.png` достаточно успешного `migrate` + `Установка завершена успешно`.
